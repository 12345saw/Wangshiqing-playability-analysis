### 方法 1：如果您已经有 GitHub 仓库

如果您已经在 GitHub 上创建了仓库（例如：`wangshiqing/layability-analysis`），请告诉我仓库地址，我会帮您连接并推送。

仓库地址格式：
- `https://github.com/您的用户名/仓库名.git`
- 或 `git@github.com:您的用户名/仓库名.git`

### 方法 2：创建新的 GitHub 仓库

1. **访问 GitHub**：https://github.com
2. **点击右上角的 "+" 按钮** → 选择 "New repository"
3. **填写仓库信息**：
   - Repository name: `league-of-legends-site`（或您喜欢的名字）
   - Description: （可选）描述您的项目
   - 选择 Public（公开）或 Private（私有）
   - **不要**勾选 "Initialize this repository with a README"（我们已经有了）
   - **不要**添加 .gitignore 或 license（我们已经有了）
4. **点击 "Create repository"**
5. **复制仓库地址**（HTTPS 格式，例如：`https://github.com/您的用户名/league-of-legends-site.git`）

### 方法 3：使用 GitHub CLI（如果已安装）

## 📝 Vercel 部署

推送完成后，在 Vercel 中：
1. 打开 Vercel 项目页面
2. 如果还没有连接 GitHub，点击 "Import Project"
3. 选择您的 GitHub 仓库
4. Vercel 会自动检测 Next.js 项目并部署

或者，如果已经连接了仓库，Vercel 会自动检测新的推送并重新部署。
