## 🚀 Vercel 部署步骤

### 方法 1：如果 Vercel 已连接 GitHub 仓库

1. **打开 Vercel 项目页面**
   - 访问：https://vercel.com/dashboard

2. **检查自动部署**
   - Vercel 会自动检测到新的推送
   - 会自动触发新的部署
   - 等待部署完成（通常 2-5 分钟）

3. **查看部署状态**
   - 在 Vercel 项目页面查看 "Deployments"
   - 确认最新部署状态为 "Ready"（绿色）

### 方法 2：如果还没有连接 Vercel

1. **访问 Vercel**
   - 打开：https://vercel.com
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击 "Add New..." → "Project"
   - 选择仓库：`12345saw/Wangshiqing-playability-analysis`
   - 点击 "Import"

3. **配置项目**
   - Vercel 会自动检测到 Next.js 项目
   - Framework Preset: Next.js（自动）
   - Root Directory: `./`（根目录）
   - Build Command: `npm run build`（自动）
   - Output Directory: `.next`（自动）

4. **部署**
   - 点击 "Deploy"
   - 等待部署完成
